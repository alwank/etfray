# ETF Discovery

**Research → Discovery** lets you browse and filter the ETF universe without knowing a ticker upfront.

## Filters

Use the horizontal filter pills above the table:

- **Asset class** — Equity, Fixed Income, Real Estate, Commodity, Multi-Asset
- **Category** — Broad Market, Sector, International, etc.
- **Geography** — US, Global, Emerging Markets, and more

Select **All** on any pill to clear that dimension.

## Search

Type a ticker or fund name in the search box and press **Enter** to query EDGAR directly (same as Research → Search).

## Table actions

- **Double-click** a row to open that ETF in the research workspace
- Use document links in the table to jump to **Documents** for that fund

## Watchlist

Add ETFs to your [Watchlist](watchlist.md) from Discovery using the watchlist control on each row (same as other research views).

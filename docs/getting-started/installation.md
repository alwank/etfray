# Installation

## Requirements

- Python 3.11 or later
- A terminal with 256-color support (most modern terminals)

## Install from PyPI

```bash
pip install etfray
```

## Install from source

```bash
git clone https://github.com/alwank/etfray.git
cd etfray
pip install -e ".[dev]"
```

## Verify

```bash
etfray
```

This launches the TUI. Press `q` to quit.
